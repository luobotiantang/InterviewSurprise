 ### SpringBoot的启动方式
     1、IDEA运行application这个类的main方法 SpringApplication.run(MySpringApplication.class,args)        
     2、在SpringBoot的应用的根目录下运行   mvn spring-boot:run
     3、使用mvn install 生成jar后运行
        先到项目根目录下
        mvn install
        cd target
        java -jar myspringboot-0.0.1-SNAPSHOT.jar
